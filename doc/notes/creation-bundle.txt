-## Créer un dossier "lib" à la racine du projet puis un dossier ""RecaptchaBundle" et enfin un fichier "RecaptchaBundle"

lib/RecaptchaBundle/RecaptchaBundle.php

-## Créer la classe RecaptchaBundle: 

<?php

namespace Igal\RecaptchaBundle;

use Symfony\Component\HttpKernel\Bundle\Bundle;

class RecaptchaBundle extends Bundle
{
    
}

-## Dans le composer.json modifier l'autoload:

"autoload": {
        "psr-4": {
            "App\\": "src/",
            "Igal\\RecaptchaBundle\\": "lib/RecaptchaBundle"
        }
    },

-## Dans le config/bundle.php ajouter notre bundle: 

\Igal\RecaptchaBundle\RecaptchaBundle::class => ['all' => true],

### CREATION DU TYPE ###

-## Dans le dossier RecaptchaBundle créer Type/RecaptchaSubmitType.php

<?php

namespace Igal\RecaptchaBundle\Type;

use Symfony\Component\Form\AbstractType;
use Symfony\Component\Form\Extension\Core\Type\TextType;
use Symfony\Component\OptionsResolver\OptionsResolver;

class RecaptchaSubmitType extends AbstractType
{
    public function configureOptions(OptionsResolver $resolver)
    {
        $resolver->setDefaults([
            'mapped' => false
        ]);
    }

    // Préfix du bloc
    public function getBlockPrefix()
    {
        return 'recaptcha';
    }

    // Spécifie le champ parent 
    public function getParent()
    {
        return TextType::class;
    }
}

-## Ajouter le champ dans notre FormType

->add('captcha', RecaptchaSubmitType::class, $this->getConfiguration("Enregistrer", ""))
